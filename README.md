# fleffy-run-2019
Fleffy Run for web browsers. It's turning out like the Sonic 06 of Fleffy games.


#Controls

Space: Jump
R: Run (when run meter says "Run")
M: Lock/unlock Fleffy's position (does not affect jumping)
